# Zen Sand

禅の砂庭をモチーフにしたブラウザゲーム。  
セルオートマトンによるリアルタイム物理シミュレーション。

## 遊び方

- **クリック / ドラッグ** でパーティクルを配置
- **ブラシスライダー** でサイズ調整
- **☁ 雨モード** をONにすると選択中のマテリアルが上から降り注ぐ

## マテリアル

| キー | 素材 | 挙動 |
|------|------|------|
| `1`  | 砂   | 重力で落下、水を押しのける |
| `2`  | 水   | 流動的に広がる |
| `3`  | 雪   | ゆっくり落下、水で溶ける |
| `4`  | 火   | 上昇して消える、雪を溶かす |
| `5`  | 壁   | 静的な障壁 |
| `0`  | 消   | セルを削除 |

## 起動方法

ファイルをブラウザで直接開くか、ローカルサーバーで起動:

```bash
npx serve .
# または
python -m http.server 8080
```

## 技術スタック

- Vanilla JS (ES Modules)
- Canvas API + ImageData (高速ピクセル描画)
- セルオートマトン (Falling Sand アルゴリズム)
- フレームワーク不使用
